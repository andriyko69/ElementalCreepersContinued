# Elemental Creepers Continued

A NeoForge port of **Elemental Creepers** for **Minecraft 1.21.1**.

**Elemental Creepers Continued** brings a collection of themed creeper variants into the game, each with its own behavior, explosion effect, or special gimmick. The goal of this port is to preserve the spirit of the original mod while making it available on NeoForge.

## Added Creepers

This port currently includes:

- Celebration Creeper
- Cookie Creeper
- Dark Creeper
- Earth Creeper
- Electric Creeper
- Fire Creeper
- Friendly Creeper
- Ghost Creeper
- Ice Creeper
- Illusion Creeper
- Light Creeper
- Magma Creeper
- Psychic Creeper
- Reverse Creeper
- Spider Creeper
- Water Creeper

## Configs

The port includes configurable behavior for the mod, and config support is being adapted for the NeoForge version.

Depending on the build, configuration may cover things like:

- enabling or disabling specific creeper types
- spawn-related settings
- explosion or effect behavior

Config files should appear in the standard Minecraft `config` folder after launching the game.
